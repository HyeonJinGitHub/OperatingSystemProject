# 개요  
* 프로세스 단기 스케줄링 알고리즘 중  RR(Round Robin),  SRT(Shortest Remaining Time), Feedback(Multiple Level Feedback Queue) 알고리즘 구현 프로그램
  * input.txt로 텍스트 형태의 입력 파일을 입력
  * 프로세스 ID, 도착시간, 서비스 시간(q) 형태로 입력
  * 프로세스 ID가 0 인 경우는 입력의 종료를 의미
    ** ex ) 1, 0, 3
    ** 2, 2, 6
    ** 3, 4, 4
    ** 4, 6, 5
    ** 5, 8, 2
    ** 0
